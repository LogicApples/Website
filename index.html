Recent Works

Cloudness Network, Minecraft Bedrock edition server, the project was born on the 1st of March 2021, a Practice server, inspired by Java Edition servers like Minemen Club but built on Bedrock platform.

Unfortunately the project failed and was shut down by XFuryus on the 27th of July 2021.

Cloud Discord Bot, project born on June 2021, the bot is used to manage our tickets and reports in the Discord Server, version one hasn't got many functions at the moment, future versions will add moderation commands.

Cloud is a private bot and will never be released to the public. We will work on public bots in the future!

Image hosting, this private project was created under XFuryus' name as XFuryus' Screenshots, the project is meant to host screenshots and videos and is inspired by upload.systems. The project is highly unstable, poor automated and still requires lot of work.

We aren't planning on making this public yet and at the moment only 2 people have access to it.
